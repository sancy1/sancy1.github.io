Enrollment Count by Course and Term

  • SQL
  • University Database Query Result (11)

The number of enrolled students in each course is obtained by this query, each and every term. The data from several tables is combined using a JOIN and groups according to course and term to determine the total number of enrolled students.

SQL CODE

SELECT c.course_title, t.term_name,
COUNT(e.student_id) AS "Total enrollment"
FROM enrollment e
JOIN section s
ON e.section_id = s.section_id
JOIN course c
ON s.course_id = c.course_id
JOIN term t
ON s.term_id = t.term_id
GROUP BY c.course_title, t.term_name;
